## Survival Phrases for Brazilian Portuguese

These phrases are organized by situation. Prioritize memorizing them as complete chunks — don't worry about the grammar behind them at first.

---

### Meta-Communication (Learn These First)

These are the most important phrases for a beginner because they keep conversations going when you get stuck.

| Portuguese | English |
|---|---|
| *Você pode falar mais devagar, por favor?* | Can you speak more slowly, please? |
| *Pode repetir, por favor?* | Can you repeat that, please? |
| *Não entendi.* | I didn't understand. |
| *O que significa...?* | What does ... mean? |
| *Como se diz ... em português?* | How do you say ... in Portuguese? |
| *Estou aprendendo português.* | I'm learning Portuguese. |
| *Meu português não é muito bom ainda.* | My Portuguese isn't very good yet. |
| *Você fala inglês?* | Do you speak English? |

### Greetings and Small Talk

| Portuguese | English | Notes |
|---|---|---|
| *Oi! / Olá!* | Hi! / Hello! | *Oi* is more common in Brazil |
| *E aí?* | What's up? | Very common informal greeting |
| *Tudo bem? / Tudo bom?* | Everything good? | Standard check-in |
| *Tudo bem, e você?* | All good, and you? | Standard response |
| *Bom dia.* | Good morning. | Used until noon |
| *Boa tarde.* | Good afternoon. | Noon to ~6pm |
| *Boa noite.* | Good evening / Good night. | After ~6pm |
| *Prazer em conhecer você.* | Nice to meet you. | |
| *Até logo. / Tchau!* | See you later. / Bye! | *Tchau* is from Italian *ciao* |
| *Até amanhã.* | See you tomorrow. | |

### Restaurants and Food

| Portuguese | English |
|---|---|
| *Uma mesa para dois, por favor.* | A table for two, please. |
| *O cardápio, por favor.* | The menu, please. |
| *O que você recomenda?* | What do you recommend? |
| *Eu gostaria de...* | I would like... |
| *Para mim, ...* | For me, ... (when ordering) |
| *Sem ... por favor.* | Without ... please. |
| *Sou alérgico/a a...* | I'm allergic to... |
| *Está delicioso!* | It's delicious! |
| *A conta, por favor.* | The bill, please. |
| *Aceita cartão?* | Do you accept card? |
| *Fica à vontade.* | Make yourself comfortable. (said by host) |

**Key food words:**
*pão* (bread), *frango* (chicken), *carne* (meat/beef), *peixe* (fish), *arroz* (rice), *feijão* (beans), *salada* (salad), *sobremesa* (dessert), *cerveja* (beer), *suco* (juice), *café* (coffee)

### Expressing Yourself

**Opinions and reactions:**

| Portuguese | English |
|---|---|
| *Que legal! / Que bacana!* | How cool! |
| *Que pena.* | What a shame. |
| *Que saudade!* | I miss (it/you) so much! *(untranslatable — deep longing)* |
| *Não acredito!* | I can't believe it! |
| *Verdade?* | Really? Is that true? |
| *Com certeza.* | Definitely. / For sure. |
| *Depende.* | It depends. |
| *Não tenho certeza.* | I'm not sure. |
| *Acho que sim. / Acho que não.* | I think so. / I don't think so. |
| *Concordo. / Discordo.* | I agree. / I disagree. |

**Expressing needs:**

| Portuguese | English |
|---|---|
| *Preciso de ajuda.* | I need help. |
| *Pode me ajudar?* | Can you help me? |
| *Não sei.* | I don't know. |
| *Esqueci.* | I forgot. |
| *Quero / Gostaria de...* | I want / I would like... |
| *Tenho fome. / Tenho sede.* | I'm hungry. / I'm thirsty. |
| *Estou cansado/a.* | I'm tired. |

### The Word *Saudade*

*Saudade* is one of the most culturally important words in Portuguese. It has no direct English equivalent — it describes a deep, bittersweet longing for something or someone you love that is absent. It can be used to describe missing a person, a place, a time in your life, or even something that may never have existed.

- *Estou com saudade de você.* — I miss you (deeply).
- *Que saudade!* — I really miss that! / Those were the days!
